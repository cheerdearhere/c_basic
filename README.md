# Visual Studio 2022 - C++ 단축키
- F5         : 디버그 모드 실행(Debugger)
- F4         : 디버그 중 에러 발생 부분 이동
- Shift + F5 : 디버그 모드 강제 종료

- F7         : 프로젝트 빌드(Compile + Link = Build)
- F9         : 중단점 설정
- F10        : 디버그 모드에서 한 행 실행
- F11        : 디버그 모드에서 함수 추정

- Ctrl + F5  : 실행(Run)
